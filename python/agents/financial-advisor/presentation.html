How the Multi-Agent System Works

📊 Data Analyst Agent

• Gathers real-time market data

• Analyzes SEC filings and news

• Provides comprehensive market overview

data_analyst_agent = Agent( model="gemini-2.5-pro-preview", name="data_analyst_agent", tools=[google_search], output_key="market_data_analysis_output" )

📈 Trading Analyst Agent

• Develops investment strategies

• Considers risk tolerance & timeline

trading_analyst_agent = Agent( model="gemini-2.5-pro-preview", instruction=TRADING_ANALYST_PROMPT, output_key="proposed_trading_strategies_output" )

⚡ Execution Analyst Agent

• Creates detailed execution plans

• Optimizes for cost and timing

execution_analyst_agent = Agent( model="gemini-2.5-pro-preview", instruction=EXECUTION_ANALYST_PROMPT, output_key="execution_plan_output" )

🛡️ Risk Analyst Agent

• Evaluates overall portfolio risk

• Provides final recommendations

risk_analyst_agent = Agent( model="gemini-2.5-pro-preview", instruction=RISK_ANALYST_PROMPT, output_key="final_risk_assessment_output" )

Technical Challenges & Solutions

🔧 Real-time Data Integration

SEC filings, market data, news sources scattered across different APIs and formats

→

✅ Google Search API Integration

Unified data gathering with automated financial information extraction and parsing

🔧 Agent State Coordination

Managing complex state between multiple AI agents and ensuring data consistency

→

✅ ADK State Management

Output keys system for seamless agent communication and data flow orchestration

🔧 Real-time User Experience

Users expect immediate feedback, but AI processing can take several seconds

→

✅ Streaming API + Fallback

Server-Sent Events for token-by-token streaming with graceful fallback to batch responses

🔧 Production Deployment

Frontend hardcoded to localhost, CORS issues, environment configuration complexity

→

✅ Environment Variables + CORS Proxy

Configurable endpoints for dev/prod with FastAPI proxy handling CORS and session management
